# Scheduling Agent Demo

Clinical Access Orchestration demo with a React scheduler portal and shared mock data fixtures.

## Structure

```
scheduling-agent/
├── mock-data/          # Compass upload bundles + scheduling fixtures
└── frontend/           # React scheduler portal
```

## Quick start

```bash
cd frontend
npm install
npm run dev
```

## Mock data

See [mock-data/README.md](mock-data/README.md) for Compass upload files and scheduling fixtures.

## Compass integration

Configure `frontend/.env` from `frontend/.env.example` when you have API credentials.
